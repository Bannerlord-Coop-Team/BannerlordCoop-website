Frequently Asked Questions

What is Bannerlord Coop?

Bannerlord Coop is a mod that is currently in development.This mod will allow you to play the campaign with your friends!

Is the mod currently playable?

No. Currently, there are a lot of unfinished features. The mod will not be playable until version 1.0

What will happen with the campaign? What about other quests?

We are using sandbox so no main quest. For the V1 release of this mod, all quests will be disabled.

How battles/scenes will work ?

The current vision of the mod is to have separate parties. With this comes the issue of world map timing while in battle. There are multiple solutions to this, our solution to this is just to simply limit or slow time on the world map. Many have suggested to just have players in the same battle but that fix is not universal enough. For example, the game pauses time in a lot more situations than just battles, like visiting a town, in that case the other player would have no one to control or at least no point in controlling them. Our suggested fix will solve this by allowing all players to continue what they were doing in any scenario.

Will everyone be in a single party or can everyone control their own?

Everyone will have a separate party. Separate parties will have more game mechanics allowing for more fun gameplay. We are planning to add a single-party mode after the release.

Will it support mods?

This will only be compatible with native at launch. After launch, we may try to work with some mods and make them compatible. Mods require further syncing.

Will players be able to make separate kingdoms and do PVP?

Yes, each player can found their own kingdom and fight each other.

How will the multiplayer work? Will there be dedicated servers? Is it Peer-to-Peer?

We will be using a dedicated/hosted client/server system. Players can connect to the server, however, the server will need to be port forwarded or all players can use a program like Hamachi to get around port forwarding.
